# SDHS AI융합실 사용신청 챗봇

이 프로젝트는 서울디지텍고등학교 2024 공간정보 소프트웨어 해커톤에서 **최우수상**을 수상한 개인 프로젝트로, 교내 **AI융합실 출입 및 사용 신청 절차를 자동화**하기 위해 개발되었습니다. 챗봇 기반의 사용자 인터페이스와 실시간 객체 인식 기술을 결합하여, 학생들의 사용 편의성과 관리 효율성을 동시에 개선했습니다.

## 🛠️ 기술 스택

- **챗봇 플랫폼**: Kakao Business
- **서버 개발**: Python, Flask
- **객체 탐지**: YOLOv5 (PyTorch 기반)
- **데이터베이스**: SQLite
- **서버 노출**: Localtunnel
- **버전 관리**: GitHub

## 🔑 주요 기능

### 1. 카카오 챗봇 기반 사용자 인터페이스

- 입실 / 퇴실 신청
- 연장 / 조기 퇴실 신청
- 사용 신청 현황 조회
- 시설 문제 제보

### 2. 실시간 인원수 탐지 기능

- YOLOv5를 활용한 **객체 탐지 모델**로 AI융합실 내부 인원 수 실시간 추적
- 출입 현황과 연동하여 자동화된 관리 가능

### 3. 자동 메시지 발송 기능

- Kakao API를 활용하여 **신청 처리 결과 자동 응답**
- 관리자에게 이상 상황 자동 보고 (ex. 초과 인원 감지 시 알림 전송)

## 🧩 시스템 구조

![챗봇 메시지 흐름](./image1.png)

- 챗봇에서 사용자의 입력을 받아 Flask 기반 스킬 서버로 전달
- 스킬 서버는 DB를 통해 사용자 상태를 저장하고, YOLO 모델과 연동된 출입 데이터 확인
- 결과는 다시 Kakao 플랫폼을 통해 사용자에게 전달

![YOLO 기반 객체 탐지](./image2.png)

- CCTV 또는 웹캠 입력을 통해 YOLO 모델이 실시간 인원 탐지 수행
- 탐지 결과는 Flask 서버에서 후처리되어 신청 시스템과 연결

## 🧠 개발 배경 및 의의

기존에는 AI융합실 출입 신청이 수기 또는 구글폼 방식으로 이뤄졌으나, 이는 다음과 같은 한계를 가지고 있었습니다:

- 출입 상황과 신청 상태가 실시간으로 연동되지 않음
- 인원 초과 여부 확인이 어려움
- 사용자 요청 처리 속도 지연

이러한 문제점을 해결하기 위해 **비대면 자동화 시스템**을 설계했으며, 사용성과 보안성을 모두 확보한 점에서 의미 있는 결과를 얻었습니다.
